# 👗 H&M Customer Purchase Pattern Analysis
> **데이터 정제부터 전략 도출까지: 단품 구매 해소 및 교차 판매(Cross-selling) 전략**

---

## 📌 Project Overview

본 프로젝트는 H&M 트랜잭션 데이터를 분석하여 **85.2%에 달하는 단품 구매 현상**을 진단하고, 연령대별 구매 패턴 분석을 통해 매출을 극대화할 수 있는 **교차 판매 전략**을 제안합니다.

단순한 데이터 요약을 넘어, 대규모 데이터 파이프라인 구축과 실무 마케팅 가이드라인 수립에 초점을 맞추었습니다.

---

## 🛠 Tech Stack

- **Language**: `Python`
- **Data Engineering**: `Pandas`, `NumPy`, `Pickle`
- **Analysis & Logic**: `itertools.combinations`, `Seaborn`, `Matplotlib`

---

## 🚀 Key Insights & Results

### 1️⃣ 데이터 클리닝 및 무결성 확보
- **나이(Age)**: 분포의 편향을 고려하여 중앙값(Median)으로 결측치를 대체했습니다.
- **이상치**: 가격(Price) 상위 0.1% 데이터를 식별하여 도메인 관점에서 검토했습니다.
- **최적화**: 데이터 타입을 보존하고 처리 속도를 높이기 위해 **Pickle(.pkl)** 형식을 도입했습니다.

### 2️⃣ 비즈니스 문제 정의: 85.2% 단품 구매율
- 전체 거래 중 약 **85.2%**가 단 1개의 상품만 구매하고 이탈함을 확인했습니다.
- 낮은 객단가(AOV) 해결을 위해 **장바구니 크기(Basket Size) 확장**을 핵심 목표로 설정했습니다.

### 3️⃣ 핵심 전략: 연령대별 연관 상품 추출
- **16-24 세대**: 비키니 상/하의 세트 구매가 압도적 → **세트 번들링 마케팅** 제안
- **35+ 세대**: 베이직 티셔츠와 슬랙스 조합 선호 → **데일리룩 패키지 추천** 제안
